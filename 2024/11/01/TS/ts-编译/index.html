步骤

  1. 在使用typeScript的文件夹下执行npm install -g typescript
  2. 新建一个.ts后缀的文件然后贴上例子
  3. 在终端执行tsc greeter.ts不报错成功
  4. 报错：以管理员身份开启Windows Powershell
  5. 输入指令：set-ExecutionPolicy RemoteSigned
  6. 输入y或a没报错执行成功
  7. 回到终端再次执行tsc greeter.ts成功
