var

  1. 存在变量提升
  2. 同一变量重复声明多次只拿到一个结果
  3. 可以在包含它的函数、模块、命名空间、全局作用域内部任何位置被访问

let 和 const

  1. 都有块级作用域
  2. 一个作用域内只能声明一次
  3. 不能在被声明之前读或写
  4. 直到声明前的区域都属于暂时性死区
  • 使用最小特权原则，所有变量除了计划去修改的都应该使用const

type 类型别名

type Second = number
// 给number这个类型命名为second，而second引用这个类型

数组解构：一一对应

  • let [, second, , fourth] = [1, 2, 3, 4]
  • 使用…语法创建剩余变量 ：
    • let [first, …rest] = [1, 2, 3, 4];
    • console.log(first,rest); // 1, [ 2, 3, 4 ]
